https://www.geeksforgeeks.org/devops/docker-docker-container-for-node-js/

Inicializa projeto nodejs
c:\> npm init

Execução na máquina local
c:\> npm run dev

Faz a construção de acordo com Dockerfile
c:\> docker build  -t docker-container-nodejs .

Faz o mapeamento entre volume do host e do container
c:\> docker run -d -p 8000:3000 -v .:/server docker-container-nodejs

Faz a execução do container com abertura do terminal
c:\> docker exec -it <Container ID> sh
