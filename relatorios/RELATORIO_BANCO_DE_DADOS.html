Resultados: Receita = qtd x preco | Custo = qtd x custoRealTotal (vw_custo_real) | Lucro = receita - custo | Margem % | Volume = qtd x volumeM3 (vw_fichas_tecnicas)

View: vw_alertas_operacionais

ColunaTipo SQLTagsDescricao

Quando gerar cada alerta:
• margem_baixa: custo_real.margemRealPct < 30% → alta; < 40% → media
• estoque_critico: estoque_mp.qtdeAtual < qtdeMinima
• vazamento_lucro: lotes.rendimentoPct < 92%
• producao_atrasada: ordens.dataPrevista < HOJE AND status='Em Producao'
• inadimplencia: contas_receber.status='Vencido'
